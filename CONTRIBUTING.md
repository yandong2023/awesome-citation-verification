# Contributing

Contributions are welcome.

Good contributions include:

- New checklists for citation verification workflows.
- Examples of metadata mismatch patterns.
- Templates for citation integrity reports.
- Tips for using public scholarly databases.
- Database-specific guidance for PubMed, Crossref, OpenAlex, Semantic Scholar, DOI.org, or publisher pages.

Please keep examples educational. Do not publish private manuscripts, private reviewer comments, or personal data.
