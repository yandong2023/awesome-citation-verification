# Checklist: Check If a Reference Is Real

Use this checklist when a reference looks plausible but needs verification.

## 1. Parse the reference

Record the visible metadata:

- Title
- Authors
- Year
- Journal or venue
- Volume, issue, pages
- DOI, PMID, PMCID, ISBN, or URL

## 2. Check identifier existence

- Paste the DOI into `https://doi.org/{doi}`.
- Search the DOI in Crossref.
- For biomedical papers, search PubMed by DOI and title.
- If there is a PMID, search PubMed directly.

If the identifier resolves to a different title, mark the citation as `metadata mismatch`.

## 3. Match metadata

Compare the reference against a trusted record:

- Title
- First author
- Publication year
- Journal name
- Volume and issue
- Page range or article number

Small formatting differences are normal. Different title, author, year, or DOI is not normal.

## 4. Check whether the source supports the claim

A real paper is not automatically a valid citation.

Ask:

- Is the cited sentence within the scope of the article?
- Is the result from the paper, or only from its introduction?
- Does the paper support causation, association, or only background context?
- Does the cited population match the claim?
- Does the cited intervention, exposure, or outcome match the claim?

## 5. Decide the status

Use one of these:

- `verified`
- `metadata mismatch`
- `not found`
- `suspicious`
- `does not support claim`
- `needs manual review`

## 6. Suggested action

- Accept verified citations.
- Replace fabricated or mismatched citations.
- Add a note when claim support is uncertain.
- Prefer primary sources when a review only summarizes evidence.
